# 🛡️ AI-Powered SOC Copilot for Threat Hunting and Incident Investigation

## 📌 Project Overview

The AI-Powered SOC Copilot is an intelligent cybersecurity solution designed to assist Security Operations Center (SOC) analysts in detecting, investigating, and responding to cyber threats. The system analyzes security logs, identifies suspicious activities using Machine Learning, and leverages Large Language Models (LLMs) to provide threat explanations and remediation recommendations through an interactive dashboard.

---

## 🎯 Objectives

- Collect and analyze security logs from multiple sources.
- Detect anomalous activities using Machine Learning.
- Assist analysts in threat hunting and incident investigation.
- Generate AI-powered explanations and mitigation recommendations.
- Visualize security alerts and incidents through a dashboard.
- Generate incident reports for security analysis.

---

## 🚀 Features

- Security Log Collection
- Log Analysis and Processing
- Machine Learning-based Anomaly Detection
- AI-Powered Incident Explanation
- Threat Severity Scoring
- Interactive Dashboard
- Incident Report Generation
